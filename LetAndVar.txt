Quiz
1. The difference between Var and Let is, when using Var it can be reassigned, redeclared, and is a function scope.
Let is a block-scope and can be reassigned but not redeclared.

2. Const is also a block-scope function, unlike var, however it cannot be reassigned or redeclared when used.

3. When using let you can reassign a variable, unlike const, once const is used it can not be reassigned or
redeclared.

4. Hoisting is the default behaviormof moving all the declarations to the top of the scope before code execution.
